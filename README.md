# Binary_Search-Tree
C Program For Binary Search Tree

This Is A C Program For Performing Operations On Binary Search Tree

The Operations Are :
  
  1. Create
  
  2. Insert
  
  3. Delete
  
  4. Find
  
  5. Find Min
  
  6. Find Max
  
  7. Inorder
  
  8. Preorder
  
  9. Postorder
  
  10. Exit
  
The GCC File Is Also Pushed
